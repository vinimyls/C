# C
A pacman game in C.

This project is part of a course about C lengauge.

Run this in linux (terminal); 
``` 
gcc fogefoge.c mapa.c ui.c -o jogo.out
```

## the game
```
w = up
s = down
a = left
d = right

b = explode bomb (kill all ghosts in three square around the hero)
```
